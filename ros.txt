安装：

	1. 参考：http://wiki.ros.org/kinetic/Installation/Ubuntu
	2. 添加软件源（最好使用国内镜像地址，速度快）：

xunyi@odroid:~$ sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ $DISTRIB_CODENAME main" > /etc/apt/sources.list.d/ros-latest.list'

	1. Set up your keys 

sudo apt-key adv --keyserver hkp://ha.pool.sks-keyservers.net:80 --recv-key 421C365BD9FF1F717815A3895523BAEEB01FA116

          这一步一直出问题，一只以为是需要翻墙的原因，折腾半天，最后发现是因为DNS没有配置好，没有网络，解决：http://blog.csdn.net/seiulaomi/article/details/64920287

	1. 在非root用户下一步步执行

sudo apt-get update
sudo apt-get install ros-kinetic-ros-base
apt-cache search ros-kinetic

sudo rosdep init
rosdep update

	1. Environment setup

echo "source /opt/ros/kinetic/setup.bash" >> ~/.bashrc
source ~/.bashrc

	1. Dependencies for building packages

sudo apt-get install python-rosinstall python-rosinstall-generator python-wstool build-essential

	1. 检查是否安装成功

问题记录：

	1. 执行sudo命令时报错：

xunyi@odroid:~$ sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ $DISTRIB_CODENAME main" > /etc/apt/sources.list.d/ros-latest.list'
[sudo] password for xunyi:
xunyi is not in the sudoers file.  This incident will be reported.
          解决方法：root权限下执行：
root@odroid:/home/xunyi# getent group sudo     // 查看可以sudo的用户
sudo:x:27:
root@odroid:/home/xunyi# sudo adduser xunyi sudo
Adding user `xunyi' to group `sudo' ...
Adding user xunyi to group sudo
Done.
root@odroid:/home/xunyi# getent group sudo     
sudo:x:27:xunyi
